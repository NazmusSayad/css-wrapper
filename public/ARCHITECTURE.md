# Public Directory

Static files served directly by Next.js.

## Structure

```
public/
├── favicon.ico          # Site icon
├── images/              # Image assets
├── icons/               # Icon files
└── manifest.json        # PWA manifest
```

## Usage

```tsx
<img src="/images/logo.png" alt="Logo" />
<link rel="icon" href="/favicon.ico" />
```
